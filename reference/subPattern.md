# Common taxonomic name substitutions

Common taxonomic name substitutions

## Usage

``` r
subPattern()
```

## Value

character vector with regular expressions for use with
[`gsub()`](https://rdrr.io/r/base/grep.html)

## Details

Used in argument `sub_pattern` in
[`correctTaxo()`](https://umr-amap.github.io/BIOMASS/reference/correctTaxo.md)
